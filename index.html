What is Java? (No, not coffee)

  • Object-Oriented Programming Language
  • One of the most popular languages

Number

Numbers are broken down into:

  • Integers: int trackNumber = 10
  • Doubles: double oneHalf = 0.5
  • Longs: long oneQuarter = 0.25
  • Shorts: short aNumber = 1
  • Floats float anotherNumber = 2000.00
  • Bytes: byte yetAnother = 50.53

Strings 

  "Java".length()  // 4 
  "Delaware".length()  // 8

Strings are sequences of Unicode characters, with each character represented by a unique numeric value that applies across platforms and programs.

If/Else Conditionals

If a, then do b.
Else , do c.

Boolean willItRainToday = false;
if (willItRainToday) {
System.out.println("Good day for a movie inside")
} else {
System.out.println("Let's go to the beach!");
}
// Result: "Let's go to the beach!"

Iteration: While Loop

Body of the loop is executed at least once.

Loop ends when the conditional is met

while (condition is true) { execute this code }

Iteration: For Loop

Lets you provide the control information on one line

for (int i = 0; i < 10; i++)
  • Create the counter
  • Provide the conditional
  • Step through the counter
for (int i = 99; i != 0; i--) { System.out.println(i + "bottles of milk on the wall" + i + "bottles of milk" + "take one down, pass it around" + i - 1 + "bottles of milk on the wall") } Result: Drinking a lot of milk

Iteration: For Each

Performs an action on all objects in a collection (Array, etc)

String[] names = ["Joe", "Jimmy"]; for (String name : names) { System.out.println("Hi " + name); } Result: "Hi Joe" "Hi Jimmy"

Objects and Classes

Objects have state and behavior

These states and behaviors are defined through the class

The class is the blueprint for the object it supports

Classes contain constructors, getters, and setters for the object

Creating an Object

    Declaring a variable with the object type

    Instantiating the variable to create the object ("new" keyword)

    Initializing the variable by calling the constructor.

public class Library { String libraryName; }
Libary bearLibrary = new Library("bearLibrary");

Constructors

Special method that performs set-up tasks for an instance of a class

Only called once in the life cycle of an instance

Named after the object class they belong to

Unique as they're functions with no return value or type

public Library(String aName, Array[Book] aList) { this.libraryName = aName; this.bookList = aList; }

Getters ∓ Setters

Used to get or set fields in an instance of an object

Required to access fields that are set to private

public String getLibraryName() { return this.libraryName; } public void setLibraryName(String name) { this.libraryName = name; }

Functions

How objects can talk to each other

Can take in parameters, but not required

Can return a value (object, String, number), but can be void

public int addFive(int aNumber) { return aNumber += 5; }

Arrays

A kind of collection of elements

Used to store a fixed set of elements

Stored by index (starts at 0)

String[] cities = ["Tokyo", "NYC"]; cities[0] // "Tokyo" cities[1] // "NYC"

HashMaps

Another kind of collection of elements

Stores elements via key-pair values

All keys must be unique

Expandable/shrinkable

Map citiesAndStates = new Hashmap<>(); citiesAndStates.put("Dover", "DE"); citiesAndStates.put("Ocean City", "MD"); citiesAndStates.get("Ocean City") // "MD" String[] mapKeySet = citiesAndStates.keySet() // {"Dover", "Ocean City"} String[] mapValueSet = citiesAndStates.valueSet() // {"MD", "MD"}
