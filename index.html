Number

Numbers are broken down into:

  • Integers: int trackNumber = 10
  • Doubles: double oneHalf = 0.5
  • Longs: long oneQuarter = 0.25
  • Shorts: short aNumber = 1
  • Floats float anotherNumber = 2000.00
  • Bytes: byte yetAnother = 50.53

If/Else Conditionals

If a, then do b.
Else , do c.

Boolean willItRainToday = false; if (willItRainToday) { System.out.println("Good day for a movie inside") } else { System.out.println("Let's go to the beach!"); } - Result: "Let's go to the beach!"

Iteration: While Loop

Body of the loop is executed at least once.

Loop ends when the conditional is met

while (condition is true) { execute this code }

Iteration: For Loop

Lets you provide the control information on one line

for (int i = 0; i < 10; i++)
  • Create the counter
  • Provide the conditional
  • Step through the counter
for (int i = 99; i != 0; i--) { System.out.println(i + "bottles of milk on the wall" + i + "bottles of milk" + "take one down, pass it around" + i - 1 + "bottles of milk on the wall") } Result: Drinking a lot of milk

Iteration: For Each

Performs an action on all objects in a collection (Array, etc)

String[] names = ["Joe", "Jimmy"]; for (String name : names) { System.out.println("Hi " + name); } Result: "Hi Joe" "Hi Jimmy"
