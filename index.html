Number

Numbers are broken down into:

  • Integers: int trackNumber = 10
  • Doubles: double oneHalf = 0.5
  • Longs: long oneQuarter = 0.25
  • Shorts: short aNumber = 1
  • Floats float anotherNumber = 2000.00
  • Bytes: byte yetAnother = 50.53

What is CSS?

    Cascading Style Sheets
  • Allows you to modify HTML elements
  • Colors, Borders, Size, and Positioning

Colors in CSS

    You can change the colors of:
  • Text: color: red
  • Background: background-color: #000000;
  • Useful Link: htmlcolorcodes.com

Borders

Edges around elements like images
Borders are determined by thickness (px), style and color Ex: A solid black border
border: 1px solid black;
Ex: A dotted red border
border: 3px dotted #FF5733;

Sizing Text in CSS

Done in units of em; relative unit to the font. Better than using px., as that can vary on different screens
font-size: 1.4em; You can make text, images, and other elements bigger/smaller

Margins and Padding

Margins: Transparent box that separates elements Padding: Space b/w border and the element
padding-left: 10px; padding-right: 10px; padding-top: 10px; padding-bottom: 10px; OR
padding: 10px 10px 10px 10px (top, right, bottom, left)
margin-left: 10px; margin-right: 10px; margin-top: 10px; margin-bottom: 10px; margin: 10px 10px 10px 10px (top, right, bottom, left)
